/////////////////////////waeHUD2.0////////////////////////////
Is supported on all resolutions, no mac support as of now


To install go to C:\Program Files (x86)\Steam\SteamApps\common\Team Fortress 2\tf\custom and drag waeHUD2.0-master into it.

For a different crosshair go to scripts\hudlayout.res and at the bottom there will be some preset crosshairs.
